## 从切片转数组

* 数组是相同数据类型组成长度固定的连续内存数据结构。GO 中的数组变量属于值类型，当数组变量被赋值或传递时，实际上会复制整个数组。
* slice只是指向底层数组的引用类型。

``` golang
// 切换转数组 使用copy函数即可：
slice := []byte("abcdefgh")
var arr [4]byte
copy(arr[:], slice[:4])
fmt.Println(arr)
```

在go的场景中，似乎不需要这样搞。
